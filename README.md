
An Assetto Corsa car swapper that lets you switch cars without losing your location.

The app saves your position in memory while the new car is loading, then restores it once the loading is complete.

Requirements:

  Content Manager must be installed.
 
  Custom Shaders Patch (CSP) is required.

this build is in developement and is not really optimized

By default, the app can be opened with F9, but you can change the hotkey in the app settings. You can also open it from the Apps section on the right side of the screen while in Assetto Corsa.

I'm not completely sure how the default settings will behave on different systems, so I'd really appreciate any feedback. If you run into any issues or have suggestions for improvements, please let me know!
